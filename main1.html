1HTML Kya Hoti Hai?

HTML ek language hai jisse hum websites banate hain. Jis tarah LEGO blocks se ghar banta hai, waise hi HTML tags se webpage banta hai.

HTML ka Full Form:

  • H ‚Äî Hyper
  • T ‚Äî Text
  • M ‚Äî Markup
  • L ‚Äî Language

HTML website ka skeleton hota hai ‚Äî ye batata hai ke heading kahan hogi, text kahan hoga, picture kahan ayegi.

üßí Bacho ke liye yaad rakhne ka tareeqa: HTML = Website ka blueprint jo baata hai kya kahan hona chahiye!

2HTML Tags Kya Hote Hain?

Tags special words hote hain jo angle brackets < > ke andar likhe jate hain.

Example Tag:

<p>Yeh mera pehla paragraph hai!</p>
  • <p> ‚Äî start tag
  • </p> ‚Äî end tag
  • Beech ka text ‚Äî paragraph

Tag bilkul magic words ki tarah kaam karte hain! Jab aap tag lagate ho to computer samajh jata hai ke kya banana hai.

üí° Yaad Rakho: Har tag ka opening aur closing hota hai ‚Äî bilkul jaise door khul-band hota hai!

3Important Basic Tags

1. <h1> ‚Äî Big Heading

Jaise kitaab me chapter ka naam bada hota hai, waise website me <h1> badi heading hoti hai.

<h1>Welcome!</h1>

2. <p> ‚Äî Paragraph

Lambe sentences ka group paragraph hota hai.

<p>Mera naam Hassan hai.</p>

3. <img> ‚Äî Image

Website me tasveer lagane ke liye.

<img src="cat.png" alt="Cute Cat">

4. <a> ‚Äî Link

Dusre page par jane ke liye.

<a href="https://google.com">Google</a>

4HTML Page Ka Structure

Har HTML page ek basic structure follow karta hai.

<!DOCTYPE html> <html> <head> <title>Meri Pehli Website</title> </head> <body> <h1>Assalam o Alaikum!</h1> <p>Yeh mera pehla webpage hai.</p> </body> </html>

Ye structure bilkul waise hai jaise school notebook me:

  • Cover page
  • Heading
  • Content
üè´ School Comparison:
  • <head> = Notebook ka cover
  • <body> = Notebook ki pages
  • <title> = Notebook ka naam

5Lists (UL, OL, LI)

Lists Kya Hoti Hain?

List ka matlab hota hai cheezon ki ginti ya points ki list. Jaise school me:

  • Bag me kya kya hona chahiye
  • Lunch box me kya hai
  • Homework ke tasks

üü£ 1. UL ‚Äî Unordered List (Bullet Points)

UL ka matlab hota hai Unordered List ‚Äî yani jisme numbering nahi hoti, bas bullet points hotay hain.

<ul> <li>Pencil</li> <li>Rubber</li> <li>Sharpener</li> </ul>

üü¢ 2. OL ‚Äî Ordered List (Numbering)

OL ka matlab hota hai Ordered List ‚Äî yani 1, 2, 3 ki form me ginti wali list.

<ol> <li>Brush karna</li> <li>Nashta karna</li> <li>School jana</li> </ol>

üü° 3. LI ‚Äî List Item

LI ka matlab hota hai List Item. Yani list ke andar jo bhi point likha jata hai, woh <li> ke andar hota hai.

üßí Yaad Rakho:
  • UL = "Unginti" ‚Üí jisme numbers nahi hotay
  • OL = "Order" ‚Üí sab kuch number ke order me
  • LI = List ka chota hero!

6Tables & Forms

PART 1 ‚Äî TABLES (School Ki Copy Jaise!)

Table woh hoti hai jisme rows aur columns hote hain. Bilkul waise hi jaise school me:

  • Attendance register
  • Marks sheet
  • Timetable

Table Tags:

  • <table> ‚Äî Table start karna
  • <tr> ‚Äî Table Row (horizontal line)
  • <th> ‚Äî Table Heading (bold)
  • <td> ‚Äî Table Data (normal cell)

Example Table:

<table> <tr> <th>Name</th> <th>Class</th> <th>Marks</th> </tr> <tr> <td>Ali</td> <td>2</td> <td>90</td> </tr> <tr> <td>Ayesha</td> <td>3</td> <td>95</td> </tr> </table>
Name Class Marks
Ali 2 90
Ayesha 3 95

PART 2 ‚Äî FORMS (Notebook Ke Blanks Jese!)

Form woh hota hai jisme users se information li jati hai. Bilkul waise hi jaise school ke admission forms, jisme blanks hotay hain.

Form Tags:

  • <form> ‚Äî Form start karna
  • <label> ‚Äî Field ka naam
  • <input> ‚Äî Text box
  • <textarea> ‚Äî Bada paragraph box
  • <button> ‚Äî Click karne wala button

Example Easy Form:

<form> <label>Name:</label> <input type="text" /> <label>Class:</label> <input type="number" /> <label>Message:</label> <textarea></textarea> <button>Submit</button> </form>
üßí Cute Explanation:
  • Form = School form jisme blanks hotay hain
  • Input = Blank line jahan student name likhta hai
  • Label = "Name", "Class" jese headings
  • Button = "Submit" jese final wala button
